# 🤖 Chatbot Conversation Analysis & User Satisfaction Visualization

A Data Analytics project that evaluates chatbot conversation logs and visualizes user satisfaction levels using Excel, Python, SQL, and Power BI.

---

## 📌 Project Overview

This project analyzes chatbot conversation logs to measure user satisfaction through sentiment analysis and interactive dashboards. The goal is to identify user sentiment, evaluate chatbot performance, and provide business recommendations to improve customer experience.

---

## 🎯 Objectives

- Analyze chatbot conversation logs.
- Perform sentiment analysis on user messages.
- Measure user satisfaction using ratings.
- Visualize insights with Power BI.
- Generate business recommendations for chatbot improvement.

---

## 🛠️ Tools & Technologies

- **Microsoft Excel** – Data Cleaning & Preprocessing
- **Python** – Data Analysis & Sentiment Analysis
- **Pandas**
- **TextBlob**
- **Matplotlib**
- **MySQL** – Data Storage & SQL Queries
- **Power BI** – Dashboard & Data Visualization

---

## 📂 Project Workflow

1. Collect chatbot conversation data.
2. Clean and preprocess the dataset.
3. Perform sentiment analysis using Python.
4. Assign satisfaction ratings.
5. Store and query data using MySQL.
6. Create interactive Power BI dashboards.
7. Analyze results and generate insights.

---

## 📊 Dashboard Features

- 😊 Sentiment Distribution
- ⭐ User Satisfaction Ratings
- 📈 Chatbot Performance Overview
- 📊 KPI Cards
- 📉 Bar Charts
- 🥧 Pie Charts
- 📋 Summary Tables

---

## 🔍 Key Insights

- Most chatbot interactions are **Neutral**.
- Positive responses indicate satisfactory chatbot performance.
- Negative interactions reveal areas for improvement.
- The chatbot provides moderate user satisfaction.
- Continuous monitoring can improve chatbot quality.
